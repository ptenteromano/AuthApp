# README

### Authentication App.
User Model is a parent class for the:
1. Admin
2. Customer

Admins are able to view and edit all Customers
Customers are only able to view themselves.

* Ruby version
- 2.7.3

* System dependencies
- rspec, et al

* Database creation
- postgres sql

* Database initialization
`rails db:seed`

* How to run the test suite
`rspec spec/`
